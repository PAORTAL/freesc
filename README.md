1. ล๊อกอิน VPS แล้ว ก๊อปลืงก์นี้ไปวางใน VPS
```
wget https://raw.githubusercontent.com/lizsvr/XRAY-MULTI/main/setup.sh && chmod +x setup.sh && ./setup.sh
```
2. หากลงรันสคริปเรียบร้อยแล้ว สามารถเข้า VPS แล้วพิมพ์

```
menu
```
